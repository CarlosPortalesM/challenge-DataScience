# challenge-DataScience
Primer challenge: Alura Store
Este proyecto tiene como objetivo analizar el desempeño de cuatro tiendas con el fin de determinar cuál presenta el rendimiento más bajo y, por lo tanto, debería ser vendida.
Tecnologías usadas:
Para resolver y analizar los requerimientos se uso Python, Pandas, Matplotlib en Google Colab
Indicadores analizados:
Ingresos totales
Calificación promedio
Costo de envío promedio
Productos destacados
Categorías destacadas
